    Class EntityProperties

    The properties of the entity. From the eucalyptus editor, you are able to make custom properties such as 'speed' or 'health'. This class allows you to create and edit new value during the runtime of the script.

    Usage:

    props.setNumber("hp", 100);
    const hp = props.getNumber("hp"); // 100
